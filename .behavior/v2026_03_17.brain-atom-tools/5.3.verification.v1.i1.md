### flaky test rationalization

**case5[t1] and case8** - tools + direct answer:
- **cause**: when tools are plugged, response_format is not sent (by design, to allow tool use)
- **effect**: if model answers directly, output may not match schema
- **documented**: assumptions review (response_format conflicts with tools)
- **verdict**: known trade-off, not a code defect

**case9 kimi/k2** - 503 errors:
- **cause**: external Together AI service unavailable for kimi/k2 model
- **effect**: test times out
- **documented**: external API issue, not a code defect
- **verdict**: retry later or skip this model
