**what do we assume here without evidence?**
- that `tools` parameter structure matches openai
- that `tool_calls` response structure matches openai
- that `role: "tool"` messages work for results

**what evidence supports this assumption?**

**did the wisher say this, or did we infer it?**

**verdict**: ✅ HOLDS — confirmed via official docs, matches our SDK usage pattern
