1. defines tools with zod schemas
2. asks the model with tools plugged
3. loops while calls exist
4. executes invocations
5. continues episode with results

each step maps directly to implementation:
- step 1 → genBrainPlugToolDeclaration from rhachet (reused, not reimplemented)

- step 5 → prompt accepts BrainPlugToolExecution[] for continuation
